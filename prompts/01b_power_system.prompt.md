# 力量体系设计提示词

## 目标
为小说世界设计一个平衡、有趣且有深度的力量体系。

你是一名系统设计师，请根据以下世界设定，为小说创建完整的力量体系：

## 输入信息
- 世界观概述：
{{world_overview}}
- 题材类型：{{genre}}
- 主角背景:{{admin}}

## 输出目标
请输出：
1. **力量体系总览**：描述世界中的能量/修炼逻辑来源（灵气、源能、科技矩阵等）
2. **境界划分表**（从低到高至少六层）
3. **修炼路径**：主角与配角常用的成长途径
4. **能量副作用与限制**：避免角色无敌
5. **成长动力核心公式**：例如「牺牲 → 感悟 → 突破」等过程
6. **力量体系的象征哲学**：体现世界精神或宿命象征

> 要求：体系必须可扩展、有内在代价，避免俗套。成长需与人物心理共振。
输出变量：power_system

## 提示内容
请基于已有的世界设定，设计一个完整的力量体系：

1. **力量体系名称**：为你的力量体系取一个合适的名称
2. **力量来源**：解释力量的根本来源（如自然元素、灵魂、科技等）
3. **能力分类**：描述不同类型的能力及其特点
4. **等级划分**：设计清晰的实力等级体系
5. **修炼方法**：说明获取和提升力量的途径
6. **限制条件**：描述使用力量的代价或限制
7. **特殊能力**：介绍一些罕见或独特的能力
8. **平衡性**：如何确保不同能力之间的平衡

请确保力量体系与世界设定紧密结合，并为故事发展提供足够的可能性。
