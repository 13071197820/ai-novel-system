# 节奏控制提示词

## 目标
设计小说的节奏变化，确保故事有张有弛、引人入胜。

## 输入信息
- 成长曲线：
{{growth_curve}}
- 故事情感基调：
{{emotional_tone}}

## 输出目标
1. **节奏分布表**（章节序号 vs 张力）
2. **情绪波动曲线说明**（波峰与波谷）
3. **张力触发点说明**（关键节点原因）
4. **节奏比例建议**：平静：紧张：爆发 ≈ 2:3:1

## 提示内容
请基于已有的情节结构，详细设计故事的节奏安排：

1. **整体节奏框架**：
   - **开场节奏**：如何吸引读者并设定基调
   - **中段节奏**：如何保持读者兴趣并推进情节
   - **结尾节奏**：如何构建高潮并收尾

2. **节奏变化点**：
   - **紧张点**：提高故事紧张感的关键场景
   - **舒缓点**：让读者放松的过渡场景
   - **悬念点**：制造悬念的关键时刻
   - **揭示点**：揭晓重要信息的时刻

3. **场景节奏**：
   - **快节奏场景**：动作、冲突激烈的场景
   - **中等节奏场景**：对话、发展的场景
   - **慢节奏场景**：描述、反思的场景

4. **章节节奏规划**：
   - 为每个主要章节设定适当的节奏
   - 设计章节之间的节奏过渡

5. **节奏与情感的关联**：
   - 如何通过节奏控制读者的情绪
   - 如何匹配角色情感与场景节奏

6. **节奏技巧**：
   - 如何使用短句加快节奏
   - 如何使用长句放缓节奏
   - 如何通过留白创造节奏变化

请确保故事节奏变化自然，既避免过长的平静，也避免持续的高强度，让读者在阅读过程中保持兴趣和投入感。
