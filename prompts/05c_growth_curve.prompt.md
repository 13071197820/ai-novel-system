# 成长曲线提示词

## 目标
为主要角色设计一个自然、有深度且令人信服的成长历程。

请为主角构建单调递增、指数加速的成长曲线。

## 输入信息
- 优化后冲突：
{{refined_conflicts}}
- 人物：
{{characters}}

## 输出目标
1. **成长阶段表**：每阶段的目标、突破、代价
2. **曲线模型说明**：线性增长到指数爆发的节点
3. **情绪驱动力**：是什么让主角不断上升
4. **主角核心领悟**：每阶段的精神成长关键词

> 输出格式为时间序列描述，可视为成长函数 narrative(t)。
输出变量：growth_curve

## 提示内容
请为故事的主要角色（尤其是主角）设计详细的成长曲线：

1. **起点状态**：角色在故事开始时的能力、性格和心态
2. **成长目标**：角色在故事结束时应达到的状态
3. **成长阶段**：
   - **初始阶段**：角色的舒适区和日常状态
   - **挑战阶段**：角色面临的第一次重大挑战
   - **挫折阶段**：角色经历的失败和低谷
   - **反思阶段**：角色的自我审视和学习
   - **转变阶段**：角色的心态和能力开始改变
   - **突破阶段**：角色克服主要障碍
   - **成熟阶段**：角色实现成长和蜕变

4. **成长催化剂**：
   - **关键事件**：促成角色改变的重要事件
   - **关键人物**：影响角色成长的重要他人
   - **关键抉择**：角色必须做出的重要选择

5. **成长代价**：角色为成长所付出的代价

6. **成长表现**：
   - **行为变化**：角色行为模式的改变
   - **思维变化**：角色思考方式的改变
   - **价值观变化**：角色价值观和信念的调整

7. **成长的真实性**：如何确保角色成长自然可信

请确保角色的成长曲线与故事情节紧密结合，并且成长过程既有进步也有挫折，展现出真实而复杂的人物发展。
