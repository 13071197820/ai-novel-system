# 角色设计提示词

## 目标
创建有深度、有魅力且立体的小说角色。
你是一名小说人物设定专家，请基于以下资料创建完整的人物群像。 配角不一定是人物，可以是动物，灵物，规则，具备灵智的实体。

## 输入信息
- 世界观：
{{world_overview}}
- 背景资料：
{{lore_background}}
- 力量体系：
{{power_system}}
- 主角性别：
{{protagonist_gender}}
- 主角背景:{{admin}}

## 输出目标
1. **主角档案**：姓名、外貌、性格核心、内在信念、成长目标、弱点
2. **反派档案**：其动机必须“合理但错位”，让读者能部分共情
3. **三百名配角设定**：分别负责「指引 / 背锅 / 对照 / 背景 / 剧情 / 切换视角」等职能，配角具备自己的人生目标和行为逻辑。
4. **核心关系网络简表**：说明核心配角各人物间的相互推动关系
5. **角色弧线**：主角与核心配角的变化曲线关键词（例：“迷茫 → 觉醒 → 坚定”）
6. **配角发展**：每个配角的成长路径和变化与剧情占比符合配角权重，确保配角在故事中扮演重要角色时会被提及能被读者感知到。
7. **配角分配**：每个配角的关系网络在背景资料中不平均分配，重要配角主要存在于主角相似度更高的组织、地图中 为了减少读者理解成本，不能平均分配。
> 要求：人物动机要有代价，角色语言/理念能体现世界精神。
输出变量：characters
