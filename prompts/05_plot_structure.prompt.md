# 情节结构提示词

## 目标
构建一个有逻辑性、节奏感强且引人入胜的故事框架。

你是一名叙事设计师，请根据以下资料生成主线剧情结构。

## 输入信息
- 世界观：
{{world_overview}}
- 人物：
{{characters}}
- 场景：
{{scenes}}

## 输出目标
1. **剧情结构：三幕五阶段**（引子 → 成长 → 转折 → 高潮 → 余波）
2. **主线冲突**：世界级危机 + 个人内心危机双线并行
3. **核心支线**：推动主角变化的关键支线
4. **哲学内核**：该故事最终在讨论什么
5. **叙事节奏点（节拍）**：每幕的“张力峰值”出现点

输出变量：plot_structure

## 提示内容
请基于已有的设定，设计完整的情节结构：

1. **故事主线**：明确故事的主要情节线
2. **副线情节**：设计1-2条与主线相关的副线
3. **三幕结构**：
   - **第一幕**：介绍、设定、引发事件
   - **第二幕**：发展、冲突升级、转折
   - **第三幕**：高潮、解决、结局
4. **情节点设计**：
   - **引发事件**：推动故事开始的关键事件
   - **第一个转折点**：改变故事方向的事件
   - **中点**：故事的平衡点，通常是角色态度转变的地方
   - **第二个转折点**：将故事推向高潮的事件
   - **高潮**：故事的最高冲突点
   - **结局**：冲突解决和故事收尾
5. **伏笔设计**：在早期埋下后期将揭晓的线索
6. **冲突设计**：设计角色面临的内部和外部冲突

请确保情节结构紧凑，节奏合适，并能够充分展现角色成长和世界特色。
