# 大纲生成提示词

## 目标
整合所有已有设定，生成一个详细、结构化的小说大纲。

## 提示内容
请基于之前创建的所有设定（世界、角色、场景、情节结构、冲突、成长曲线和节奏安排），生成一个完整的小说大纲：
## 输入信息
{{meme_adjusted_outline}}

## 输出目标
1. **每章一句话概述（不少于500章）**
2. **每幕主题关键词**
3. **转折点与牺牲点标记**
4. **主角成长节点映射**

> 要求：情节推进线性上升，节奏紧凑，结尾具悬念。
输出变量：outline

1. **大纲结构**：
   - **卷/部划分**：如果适用，划分小说的主要部分
   - **章节规划**：详细规划每一章的内容
   - **场景安排**：将之前设计的场景分配到各个章节

2. **章节大纲**：
   - 为每一章提供：
     - **章节标题**：简洁的章节名称
     - **主要场景**：本章包含的关键场景
     - **主要事件**：本章发生的重要事件
     - **角色发展**：角色在本章的变化
     - **冲突发展**：冲突的推进
     - **悬念设置**：为本章结尾设置的悬念

3. **细节标注**：
   - 在大纲中标注需要特别注意的细节
   - 标注需要埋下伏笔的地方
   - 标注需要呼应前文的地方

4. **节奏指示**：
   - 为每一章标注建议的节奏
   - 标注高潮和舒缓部分

5. **主题强化**：
   - 标注各章节如何强化故事的核心主题
   - 确保主题在整个故事中贯穿一致

6. **连贯性检查**：
   - 确保情节发展逻辑连贯
   - 确保角色行为符合其性格和动机
   - 确保世界设定在整个故事中保持一致

请生成一个既详细又灵活的大纲，为后续的章节写作提供清晰的指导，同时保留足够的创作空间。
